# Testes com RSPEC

Este é um guia rápido sobre como usar o RSPEC para escrever testes em Ruby.

## Instalação

Adicione o RSPEC ao seu Gemfile:

```ruby
gem 'rspec'
```

Então execute:

```ruby
bundle install
```
